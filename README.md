# react-loader-animate-docs

Documentation site for [react-loader-animate](https://www.npmjs.com/package/react-loader-animate), deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

Push to `main` → GitHub Actions automatically builds and deploys to GitHub Pages.

**Required setup:**
1. Go to repo **Settings → Pages**
2. Set **Source** to **GitHub Actions**
3. Push to `main` — the workflow handles the rest.

## Notes

- This is a **docs-only** project. It imports `react-loader-animate` from npm.
- Source code for the library lives at [react-loader-animate](https://github.com/YOUR_USERNAME/react-loader-animate).
